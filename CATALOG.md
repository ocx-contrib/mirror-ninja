# Ninja

Ninja is a small build system with a focus on speed. It is designed to run
builds as fast as possible by being driven by higher-level build systems —
such as CMake, Meson, or gn — that generate its `build.ninja` input files.
Where other build systems are high-level languages, Ninja aims to be an
assembler: it does the minimum work to get a fast, incremental build.

## What's included

- **ninja** — the build executor that runs `build.ninja` files
